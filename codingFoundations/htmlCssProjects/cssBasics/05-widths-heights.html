Width and Height Notes

The width and heigh properties are used to set the width and height of an element.

The default setting for width and height is auto, which means the browser decides the width and heigh.

Other settings for width and height are:

  • Specified length values such as px & em
  • Percentage of the containing element (ex: 50%, 100%, etc.)
